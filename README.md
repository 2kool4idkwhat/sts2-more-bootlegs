# More Bootlegs mod

This is a Slay the Spire 2 mod that adds more bootleg relics to the Fake Merchant's pool

## Installing

Generally, I don't provide precompiled binaries for my projects, so you'll need to build the mod from source

Install .NET 9 SDK, then run `dotnet build` to compile the mod. This should also automatically install the mod

This mod depends on [BaseLib](https://github.com/Alchyr/BaseLib-StS2), so you'll need to install that too if you haven't already
